Giới thiệu

Sinh viên chuyên ngành Kỹ thuật phần mềm tại Đại học FPT định hướng trở thành kỹ sư Backend chuyên sâu kết hợp năng lực ứng dụng Trí tuệ nhân tạo (AI). Đam mê nghiên cứu và thiết kế các kiến trúc hệ thống dịch vụ có độ chịu tải cao, tối ưu hóa toàn diện hiệu suất xử lý dữ liệu lớn (Big Data SQL), cấu trúc hạ tầng linh hoạt và tích hợp liền mạch các mô hình học máy (Machine Learning/Deep Learning) vào sản phẩm thương mại thực tế. Khả năng tự học cao, tư duy hệ thống mạch lạc và luôn hướng tới việc giải quyết các bài toán tự động hóa thông minh nhằm tối ưu chi phí vận hành doanh nghiệp.

Dự án thực tế (Bấm để xem chi tiết)
Hệ Thống Quản Lý API Bán Hàng E-Commerce Enterprise Dự án cá nhân
Kiến trúc hệ thống Backend toàn diện, bảo mật và tối ưu hóa xử lý đồng thời
  • Thiết kế kiến trúc: Xây dựng hệ thống Backend phân tầng chuẩn RESTful API phục vụ luồng nghiệp vụ phức tạp của sàn TMĐT: Quản lý vòng đời sản phẩm đa thuộc tính, quy trình xử lý đơn hàng đa trạng thái, quản lý kho hàng và giỏ hàng thời gian thực.
  • Bảo mật & Phân quyền: Thiết kế phân hệ Identity Provider (IdP) sử dụng Spring Security cấu hình cơ chế Stateless Authentication qua JSON Web Token (JWT). Triển khai phân quyền theo vai trò (RBAC) chặt chẽ đến từng Endpoint kết hợp mã hóa bất đối xứng mật khẩu người dùng.
  • Tối ưu hóa Cơ sở dữ liệu: Thiết kế cơ sở dữ liệu SQL Server chuẩn hóa dạng 3NF. Trực tiếp giải quyết bài toán thắt nút cổ chai hiệu năng (Bottleneck) bằng cách cấu hình Index (Clustered/Non-Clustered) hợp lý, tối ưu các câu lệnh phức tạp chứa nhiều mệnh đề JOIN/Subquery, giảm thời gian phản hồi (Response Time) của các API thống kê báo cáo từ 4s xuống dưới 200ms trên tập dữ liệu hàng triệu bản ghi giả lập.
  • Quản trị giao dịch & Ràng buộc: Áp dụng cơ chế cô lập giao dịch (Transaction Isolation Levels) và xử lý lỗi (Global Exception Handling) đồng bộ nhằm đảm bảo tính toàn vẹn dữ liệu (ACID) tuyệt đối khi xảy ra tranh chấp tài nguyên hệ thống (Ví dụ: Trừ số lượng tồn kho khi nhiều user đặt hàng đồng thời).
Công nghệ sử dụng: Java Core (17), Spring Boot, Spring Security, Hibernate/Spring Data JPA, SQL Server, JWT, Maven, RESTful API.
Mô Hình Phân Loại Và Dự Đoán Dữ Liệu Thông Minh Enterprise Dự án nghiên cứu
Pipeline khai phá dữ liệu, huấn luyện mô hình Machine Learning giải quyết bài toán dự báo
  • Xử lý dữ liệu thô: Xây dựng đường ống (Pipeline) thu thập và làm sạch tập dữ liệu lớn. Xử lý triệt để hiện tượng dữ liệu khuyết thiếu (Missing values), lọc bỏ nhiễu (Outliers), mã hóa biến phân loại (One-Hot Encoding, Label Encoding) và chuẩn hóa thang đo dữ liệu (Standardization/MinMax Scaling).
  • Kỹ nghệ đặc trưng: Áp dụng các kỹ thuật trích chọn đặc trưng quan trọng thông qua ma trận tương quan (Correlation Matrix) và phân tích tầm quan trọng của biến để giảm chiều dữ liệu, loại bỏ hiện tượng đa cộng tuyến, giúp mô hình tập trung vào các tín hiệu dữ liệu cốt lõi.
  • Huấn luyện & Thực nghiệm: Thử nghiệm song song nhiều thuật toán học máy từ cơ bản đến phức tạp như Logistic Regression, Random Forest, Support Vector Machine (SVM) và Gradient Boosting để đánh giá bài toán phân loại/dự đoán xu hướng hành vi người dùng.
  • Tối ưu hóa siêu tham số: Triển khai kỹ thuật GridSearchCV kết hợp K-Fold Cross Validation nhằm tinh chỉnh các siêu tham số (Hyperparameters) của mô hình, ngăn chặn hiện tượng quá khớp (Overfitting) hoặc dưới khớp (Underfitting). Nâng chỉ số Accuracy và F1-Score trên tập dữ liệu kiểm thử đạt mức tối ưu cao (>90%), sẵn sàng xuất bản cấu trúc mô hình dạng file đóng gói phục vụ tích hợp ứng dụng thực tế.
Công nghệ sử dụng: Python, Jupyter Notebook, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, Google Colab.
Kỹ năng & Công cụ chuyên sâu
Ngôn ngữ lập trình
Thành thạo lập trình hướng đối tượng (OOP) vững chắc với Java (Đa luồng, Collections API, Stream API, Functional Programming). Làm chủ ngôn ngữ Python phục vụ cấu trúc kịch bản tự động hóa, thao tác cấu trúc cấu trúc ma trận dữ liệu mảng. Có kiến thức cơ bản về SQL Script vững vàng.
Framework Backend
Kinh nghiệm thực hành sâu sắc với hệ sinh thái Spring Boot (Spring MVC, Spring Security, Spring Data JPA). Hiểu sâu về cách thức hoạt động của IoC Container, Dependency Injection, Bean Lifecycle và cơ chế ánh xạ thực thể Object-Relational Mapping (ORM) qua Hibernate. Thiết kế cấu trúc hệ thống phân tầng Controller-Service-Repository chuẩn chỉ.
Cơ sở dữ liệu lớn
Tư duy thiết kế cơ sở dữ liệu quan hệ tối ưu. Thành thạo việc tạo lập lược đồ dữ liệu, cấu hình ràng buộc toàn vẹn. Kỹ năng chuyên sâu về phân tích truy vấn (Execution Plan), tối ưu hóa tốc độ câu lệnh Indexing, viết Stored Procedures, Triggers phức tạp. Khả năng làm việc tốt với SQL Server, MySQL và đang tiếp cận cơ sở dữ liệu NoSQL (Redis, MongoDB).
AI & Machine Learning
Nắm vững vòng đời phát triển của một dự án Trí tuệ nhân tạo (CRISP-DM). Thành thạo các thư viện khoa học dữ liệu chuyên dụng: Pandas, NumPy, Scikit-learn. Hiểu sâu thuật toán phân loại, hồi quy, phân cụm và các chỉ số đo lường hiệu năng mô hình (Accuracy, Precision, Recall, F1-Score, ROC-AUC). Định hướng phát triển sâu về mảng triển khai kiến trúc hệ thống AI (MLOps).
